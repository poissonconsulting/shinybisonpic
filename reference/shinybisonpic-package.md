# shinybisonpic: Shiny App for Visualizing and Cleaning Wood Bison Camera Trap Data

This app is a wrapper on the \`bisonpictools\` R package which provides
non R user's a way to visualize wood bison camera trap data. Camera trap
data is uploaded and then visualized spatially by viewing the locations
on a map and being able to explore the ratios between different
combinations of sex and life stages.
